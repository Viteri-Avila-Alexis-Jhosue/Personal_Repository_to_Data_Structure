Manual de Ayuda

Bienvenido al manual de usuario del Sistema de Parqueaderos. Este sistema le permite gestionar la información de los coches que ingresan al parqueadero de forma sencilla y eficiente.

Funcionamiento del Sistema

El sistema permite gestionar los datos de los coches en el parqueadero de manera sencilla. A continuación, se explica cómo funciona:

1. Insertar Datos de un Coche

Para registrar un coche en el parqueadero, el sistema le pedirá los siguientes datos:

  • Placa: El sistema le pedirá ingresar la placa del coche. Asegúrese de escribirla correctamente, siguiendo el formato adecuado.
  • Marca: Ingrese la marca del coche.
  • Color: Ingrese el color del coche.
  • Modelo: Ingrese el modelo del coche.

Una vez ingresados los datos, el sistema registrará el coche en el parqueadero. Si hay algún error con la placa (por ejemplo, si está mal escrita o no es válida), el sistema le pedirá que la ingrese nuevamente.

Sintaxis de la Función de Inserción de Datos:

Coche InsertarDatos() {
    placa = validador.ingresarPlaca(); 
    marca = validaciones.ingresarString("Ingrese la marca: ");
    color = validaciones.ingresarString("Ingrese el color: ");
    modelo = validaciones.ingresarString("Ingrese el modelo: ");
    return Coche(placa, modelo, color, marca);
}

Esto significa que el sistema le pedirá cada dato uno por uno, y luego creará un registro con todos los datos del coche.

2. Validación de la Placa

La placa del coche es muy importante, por lo que el sistema valida que esté escrita correctamente. Se asegura de que la placa tenga el formato adecuado, que contenga letras y números, y que la provincia sea válida.

Ejemplo de formato correcto de la placa: ABC1234.

3. Ingreso de Datos Adicionales

Para ingresar otros datos, como la marca, color y modelo del coche, el sistema le permitirá escribirlos de manera sencilla, y puede borrar cualquier error utilizando la tecla de retroceso.

Una vez que todos los datos estén completos, el coche se registrará automáticamente en el parqueadero.

4. Funciones Principales

Las principales funciones del sistema son:

  • Registrar un coche: El sistema agrega un coche con la placa, marca, modelo y color proporcionados.
  • Buscar un coche: Puede buscar un coche por su placa, y el sistema le mostrará la información del coche registrado.
  • Eliminar un coche: Si un coche ya no está en el parqueadero, puede eliminarlo utilizando su placa.

Funcionamiento de los Backups con Cobian Reflector

Para mantener la seguridad de sus datos y prevenir la pérdida de información valiosa, el sistema realiza automáticamente copias de seguridad de los archivos relacionados con el parqueadero. Esto incluye los archivos de texto donde se almacenan los datos de los coches registrados.

1. ¿Qué Archivos se Respaldan?

El sistema realiza copias de seguridad de los archivos .txt que contienen la información de los coches registrados en el parqueadero. Estos archivos son vitales para mantener la integridad de los datos, por lo que se respaldan de manera periódica.

2. Selección de los Archivos para el Backup

Cobian Reflector está configurado para seleccionar automáticamente los archivos relevantes del sistema. Los archivos de datos del parqueadero, típicamente almacenados con la extensión .txt, son los que se respaldan.

3. ¿Cómo se Crean los Backups?

El proceso de creación de backups con Cobian Reflector es automático. Sigue estos pasos:

  1. Identificación de Archivos: Cobian Reflector identifica los archivos relevantes.
  2. Creación del Backup: Se realiza una copia exacta de estos archivos.
  3. Almacenamiento del Backup: La copia de seguridad se guarda en una ubicación segura.
  4. Notificación: Cobian Reflector puede enviar una notificación indicando que el proceso de copia de seguridad se ha completado correctamente.

4. ¿Dónde se Guardan los Backups?

Los backups se guardan en una carpeta de destino que puede ser configurada durante la instalación del sistema. El destino más común es una carpeta local en su computadora, pero también puede configurar una unidad externa o una carpeta en red para almacenar las copias de seguridad, lo que proporciona una mayor seguridad ante fallos del sistema.

5. ¿Qué Hacer en Caso de Necesitar Restaurar los Datos?

Si alguna vez necesita recuperar la información de los coches registrados, simplemente abra Cobian Reflector, seleccione la tarea de backup correspondiente y elija la opción de restauración. El sistema reemplazará los archivos actuales con los datos de la copia de seguridad, asegurando que toda la información se recupere tal como estaba.
