Manual de Ayuda - Gestor de Rutas

Bienvenido al manual de usuario del Gestor de Rutas. Este sistema le permite gestionar rutas, ubicaciones y niveles de tráfico en una ciudad, así como calcular rutas óptimas y generar gráficas estadísticas.

Menú Principal

El sistema cuenta con las siguientes opciones en el menú principal:

  • Definir nueva ubicación: Permite agregar nuevas ubicaciones al sistema.
  • Definir nueva ruta: Permite crear nuevas rutas entre ubicaciones existentes.
  • Modificar ruta existente: Permite editar rutas ya creadas.
  • Definir niveles de tráfico: Permite establecer niveles de tráfico para las rutas.
  • Modificar niveles de tráfico: Permite ajustar los niveles de tráfico existentes.
  • Cálculo de rutas óptimas: Calcula la ruta más eficiente entre dos puntos.
  • Imprimir rutas: Muestra las rutas disponibles en el sistema.
  • Imprimir ubicaciones: Muestra las ubicaciones registradas.
  • Imprimir gráficas: Genera gráficas estadísticas de las rutas.
  • Ayuda: Muestra este manual de ayuda.
  • Salir: Cierra el sistema y usa un destructor para limpiar memoria usada en el programa.

Funcionalidades del Sistema

1. Definir una nueva ubicación

Para definir una nueva ubicación, ingrese los siguientes datos:

  • Nombre: Nombre del lugar. Ejemplo: Plaza Central.
  • Coordenada X: Coordenada en el eje horizontal.
  • Coordenada Y: Coordenada en el eje vertical.

Las ubicaciones se almacenan en un árbol kd para facilitar su búsqueda y gestión.

2. Definir una nueva ruta

Para crear una nueva ruta, siga estos pasos:

  1. Nombre de la ruta: Asigne un nombre a la ruta. Ejemplo: Av. Principal.
  2. Ingresar coordenadas: Conecte la ruta ingresando pares de coordenadas (x, y) que representan los puntos de la ruta.
  3. Definir velocidad y distancia: Para cada par de coordenadas conectadas, defina:
    • Velocidad máxima: Velocidad permitida en ese tramo de la ruta.
    • Distancia: Distancia entre las coordenadas. Puede ser calculada automáticamente por el sistema o ingresada manualmente.
  4. Repetir el proceso: Continúe agregando pares de coordenadas y definiendo velocidad y distancia hasta que la ruta esté completamente conectada.

Nota: Asegúrese de que las coordenadas ingresadas correspondan a ubicaciones previamente definidas en el sistema.

3. Modificar una ruta existente

En el menú secundario de modificación de rutas, puede:

  • Cambiar el nombre de la ruta. Ingresar el nombre actual de la ruta y luego el nuevo nombre.
  • Modificar la velocidad en un tramo. Deberá escribir el nombre de la ruta y las coordenadas en las que se encuentra ese tramo de ruta.
  • Modificar la distancia entre ubicaciones de un tramo. La distancia no puede ser menor a la distancia recta entre las dos ubicaciones.
  • Ampliar ruta. Permite aumentar tramos a una ruta existente , esto obiamente debe hacerse por los extremos.
  • Eliminar ruta. Aqui hay dos opciones: Eliminar ruta completa. y Eliminar tramos de una ruta. (La eliminacion por tramos se debe realizar por los etremos de la ruta).

4. Definir niveles de tráfico

Para definir niveles de tráfico en una ruta, siga estos pasos:

  1. Seleccionar la ruta: Elija la ruta en la que desea definir el nivel de tráfico.
  2. Seleccionar el tramo: Especifique el tramo de la ruta afectado por el tráfico.
  3. Definir el horario: Ingrese la hora de inicio y la hora de finalización del tráfico.
  4. Seleccionar el nivel de tráfico: Elija entre:
    • Bajo: Tráfico leve, sin retrasos significativos.
    • Medio: Tráfico moderado, con retrasos ocasionales.
    • Alto: Tráfico intenso, con retrasos considerables.

Nota: Los niveles de tráfico afectan el cálculo de rutas óptimas, especialmente en términos de tiempo estimado de viaje.

5. Modificar niveles de tráfico

Para modificar los niveles de tráfico existentes, siga estos pasos:

  1. Seleccionar la ruta: Elija la ruta con el nivel de tráfico que desea modificar.
  2. Seleccionar el tramo: Especifique el tramo de la ruta que desea ajustar.
  3. Modificar el horario: Actualice la hora de inicio y la hora de finalización del tráfico.
  4. Cambiar el nivel de tráfico: Ajuste el nivel de tráfico (bajo, medio, alto) según sea necesario.

6. Cálculo de rutas óptimas

El sistema utiliza algoritmos como Dijkstra para calcular rutas óptimas basadas en:

  • Distancia total. A veces la ruta más corta no es la más óptima, debido a los niveles de tráfico.
  • Tiempo estimado de viaje. Esta es la ruta óptima por excelencia.

También se pueden encontrar rutas alternativas no óptimas utilizando backtracking.

7. Gráficas estadísticas

El sistema genera gráficas estadísticas utilizando MatLab, incluyendo:

  • Gráficas de barras de rutas posibles (tiempo y distancia). Solamente de las 5 primeras rutas posibles.
  • Visualización de aristas (calles) y nodos (ubicaciones). El mapa se representará mediante un grafo.

8. Almacenamiento de datos

Los datos se guardan en archivos de texto (.txt) para:

  • Ubicaciones. Líneas simples que contienen coordenadas y nombre del lugar.
  • Rutas. Se guardan por tramos.
  • Niveles de tráfico. En el mismo archivo que Rutas.
