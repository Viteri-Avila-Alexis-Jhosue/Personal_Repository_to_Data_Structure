Manual de Ayuda

Bienvenido al manual de usuario del Gestor de Rutas. Este sistema le permite gestionar la informacion sobre rutas y ubicaciones en un plano bidimencional.

Funcionamiento del Sistema

El sistema controla La insercion de ubicaciones en un plano bidimencioal, es decir, cada ubicacion debe tener una coordenada "x" y una coordenada "y" .Ademas, las rutas deben estar conectadas entre ubicaciones, por lo que, entre mas puntos de referencia, mas acercado a la realidad sera el mapa de las rutas .

1. Definir una nueva ubicacion

El primer paso es definir una ubicacion, sin ubicaciones no se pueden crear las rutas, antes de comenzar a ingresar rutas, es recomendable haber ingresado todos los puntos de referencia. Los datos que tiene cada Ubicacion son:

  • Nombre: Nombre del lugar. Ejemplo: Casa
  • Coordenada x : Se refiere a la coordenada del lugar en el eje horizontal.
  • Coordenada y : Se refiere a la coordenada del lugar en el eje vertical.

Es necesario definir por lo menos 2 ubicaciones para definir rutas.

2. Definir una ruta

Despues de haber creado por lo menos 2 ubicaciones podemos continuar con el ingreso de rutas, los elementos de cada ruta son:

  • Nombre : Se refiere al nombre de la ruta. Ejemplo: Av. Simon Bolivar.
  • Velocidad maxima : Se refiere a la velocidad maxima a la que puede circular un vehiculo liviano en este tramo de la ruta.
  • Ubicaciones : Se selecciona ubicaciones ya ingresadas para definir la ruta.Este paso es definido por el numero de ubicaciones registradas pudiendose construir una ruta que abarque todas las ubicaciones.
  • Distancia : En cada tramo se define una distancia, esta puede ser calculada por el sistema o puede definirse manualmente siempre y cuando no sea menor a la distancia minima del sistema.

3. Búsqueda de Autos

El sistema permite buscar un auto por su placa y verificar su estado actual.

4. Gestión de Propietarios

Para agregar un auto nuevo, primero se debe registrar un propietario con los siguientes datos:

  • Nombre
  • Apellido
  • Cédula
  • Correo

Una vez registrado, el propietario podrá vincular autos.

5. Manejo del espacio en el parqueadero

El sistema permite visualizar el estado actual del parqueadero y la disponibilidad de espacios.

6. Busquedas Avazadas

El sistema permite realizar busquedas avanzadas de autos por placa, marca, color, modelo, rango de fechas y rango de horas .

7. Busquedas espaciales

El sistema cuenta con:

  • Busqueda del vehiculo mas cercano a una coordenada
  • Busqueda de la celda vacia mas cercana a la entrada o a la salida

Funcionamiento de los Backups con Cobian Reflector

Para mantener la seguridad de sus datos y prevenir la pérdida de información valiosa, el sistema realiza automáticamente copias de seguridad de los archivos relacionados con el parqueadero. Esto incluye los archivos de texto donde se almacenan los datos de los coches registrados.

1. ¿Qué Archivos se Respaldan?

El sistema realiza copias de seguridad de los archivos .txt que contienen la información de los coches registrados en el parqueadero. Estos archivos son vitales para mantener la integridad de los datos, por lo que se respaldan de manera periódica.

2. Selección de los Archivos para el Backup

Cobian Reflector está configurado para seleccionar automáticamente los archivos relevantes del sistema. Los archivos de datos del parqueadero, típicamente almacenados con la extensión .txt, son los que se respaldan.

3. ¿Cómo se Crean los Backups?

El proceso de creación de backups con Cobian Reflector es automático. Sigue estos pasos:

  1. Identificación de Archivos: Cobian Reflector identifica los archivos relevantes.
  2. Creación del Backup: Se realiza una copia exacta de estos archivos.
  3. Almacenamiento del Backup: La copia de seguridad se guarda en una ubicación segura.
  4. Notificación: Cobian Reflector puede enviar una notificación indicando que el proceso de copia de seguridad se ha completado correctamente.

4. ¿Dónde se Guardan los Backups?

Los backups se guardan en una carpeta de destino que puede ser configurada durante la instalación del sistema. El destino más común es una carpeta local en su computadora, pero también puede configurar una unidad externa o una carpeta en red para almacenar las copias de seguridad, lo que proporciona una mayor seguridad ante fallos del sistema.

5. ¿Qué Hacer en Caso de Necesitar Restaurar los Datos?

Si alguna vez necesita recuperar la información de los coches registrados, simplemente abra Cobian Reflector, seleccione la tarea de backup correspondiente y elija la opción de restauración. El sistema reemplazará los archivos actuales con los datos de la copia de seguridad, asegurando que toda la información se recupere tal como estaba.
